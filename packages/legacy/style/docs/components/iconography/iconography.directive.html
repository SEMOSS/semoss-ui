Iconography

SEMOSS uses Font Awesome 4 and custom icons/images. In most cases, try to use FontAwesome icons.

Best Practices

vs vs

  • (fa fa-times)
    • Deleting/Removing items. In most cases, use the x icon over the trash icon.
    • Closing windows/popovers/overlays
  • (fa fa-trash-o)
    • Deleting items permanently
    • Use the trash icon over the x icon when (fa fa-pencil-square-o) is also used
  • (fa fa-minus)
    • Minimizing windows
    • Do NOT use for deleting/removing items

Red vs Black

  • Red
    • Something is being permanently deleted
    • The icon is used with (fa fa-plus) or (fa fa-check) icon
  • Black
    • Closing windows/overlays/modals
    • Something is being temporarily removed and can be readded
